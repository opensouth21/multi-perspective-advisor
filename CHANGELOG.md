# 版本变更日志

## [v1.2.0] - 2026-06-22
- **重大变更**：本包变为**自包含（self-contained）**，用户无需额外安装 Perspective 即可正常使用
- **新增**：`perspectives/` 目录，包含 15 个 Perspective 的精简版 SKILL.md（14 位大师 + 1 位 X 运营导师）
- **更新**：视角加载优先级调整为：(1) 本包自带 → (2) Skill 工具 → (3) 用户本地路径 → (4) 内置模拟
- **新增**：激活方式增加"方法 2：从本包读取"（打包版专用）
- **新增**：激活方式方法 3 改为"模拟角色"（兜底策略）
- **修复**：解决了之前需要用户额外安装 14 个 Perspective 才能使用的依赖问题

## [v1.1.0] - 2026-06-18
- **新增**：首屏钩子（一句话定位 + 适合场景 + 你需要什么）
- **新增**：兼容性声明（纯提示语文档，兼容 Claude Code、Codex、OpenCode、OpenClaw、Hermes、WorkBuddy 等）
- **新增**：完整输出示例（投资决策场景的真实运行片段）
- **新增**：反向约束（事实查询拒绝 + 用户不满处理）
- **新增**：Perspective 路径多级探测（不再硬编码 WorkBuddy 路径）
- **变更**：视角激活方式改为四级优先级（Skill 工具 → 项目目录 → 用户常见路径 → 内置模拟）
- **新增**：失败模式表增加 2 个条目（事实查询拒绝、用户不满处理）

## [v1.0.1] - 2026-06-17
- **变更**：视角选择范围从 2-4 个调整为 4-8 个
- **新增**：强制要求至少包含 1 个跨领域视角
- **更新**：所有建议组合、输出格式、快速参考卡片

## [v1.0.0] - 2026-06-17
- 初始版本创建
- 整合 14 个 Perspective 技能
- 支持 4-8 个视角的选择范围
